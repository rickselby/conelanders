Dirt Rally

Dirt Rally results are pulled in automatically from the dirtgame.com website. This runs on a 3-hour schedule, and does an extra pull of results a few minutes before an event is due to close - in case the results completely dissappear when the event ends (which they do on the league webpage, but we can sometimes get them for a short while afterwards.

The points system was created by Georgin, which I have brought across.

The only manual steps are to add new championships, seasons, and events.

Assetto Corsa

Assetto Corsa results are generated by uploading the json results file from the server. Th3HolyMoose wrote a script to parse the file and get the results, which then feeds into the website for displaying.

The colours for the driver numbers should be representative of the car colours used in-game, but if the in-game colours don't make for a legible number, they are tweaked.

The gray background on the lap charts denotes when a driver has been lapped. However, the server lets everyone finish the whole race (not just the lap they're on once the leader finishes), so they look a little weird at the end.

  • www.assettocorsa.net
